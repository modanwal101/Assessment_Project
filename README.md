# Yatharth Group of Institution Website

## Project Overview

This is a modern and responsive educational website developed using HTML, CSS, and JavaScript. The website is designed for Yatharth Group of Institution and provides information about courses, admissions, and campus facilities.

## Features

* Animated Welcome Screen
* Responsive Navigation Bar
* Mobile Hamburger Menu
* Hero Section with Call-to-Action Buttons
* Interactive Course Cards
* Admission Form with Validation
* Light, Dark, and Cyber Themes
* Counter Animation
* Typing Effect
* Back to Top Button
* Virtual Campus Tour Slider
* Fully Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Git & GitHub

1. Download or Clone the Repository
2. Open the project folder in VS Code
3. Open index.html using Live Server
4. Explore the website in your browser
